Enter your configuration here using JSON.

Sample configuration: 

      {
        "newWindow": true,
        "repos": [
          {
            "jiraHost": "https://example.atlassian.net",
            "name": "user/repository",
            "ticketPrefixes": [
              "DEV",
              "QA"
            ]
          }
        ]
      }
newWindow
Whether or not to open links to JIRA in a new window.
repos
The list of repositories to configure.
repos[].jiraHost
The base URL to the JIRA installation, including protocol (http, https).
repos[].name
The name of the repository. If your repository lives at https://github.com/user/repository, then the name is user/repository.
repos[].ticketPrefixes
The prefixes used for tickets in JIRA for the corresponding projects. If your tickets look like DEV-12345, then your prefix for that project is DEV.
